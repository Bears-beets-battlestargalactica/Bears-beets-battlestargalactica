## 🚀 Featured Projects

### 🛡️ Provenance-Based Intrusion Detection Dataset from Operational SOC Telemetry

A research project focused on deriving a machine-learning-ready intrusion detection dataset from real operational SOC telemetry collected at ORTSOC.

- Built a provenance-oriented dataset from **Elastic EDR / Elastic Common Schema (ECS)** telemetry
- Transformed endpoint events into **CDM-like provenance graphs** using timestamp normalization, identifier stabilization, causal edge inference, and temporal graph windowing
- Evaluated provenance-based intrusion detection systems including **MAGIC, Orthrus, and ThreaTrace**
- Studied how temporal window sizes affect detection performance under realistic SOC class imbalance
- Found that PIDS models evaluated on benchmark datasets such as **DARPA Transparent Computing** do not directly transfer to noisy operational SOC telemetry

📄 **Paper:** [Deriving a Research Dataset from Operational SOC Telemetry](YOUR_PIDS_PAPER_LINK_HERE)

**Focus Areas:** Provenance Graphs, SOC Telemetry, Elastic EDR, ECS-to-CDM Mapping, Intrusion Detection, DARPA Transparent Computing
